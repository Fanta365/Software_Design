첫번째 실행 방법
1. Common_sense_Quiz.exe파일과 같은 경로에 QuizSource(.txt)파일을 위치시킨다.

2. Common_sense_Quiz.exe파일 실행, 메뉴를 선택한다.

3. 1번 메뉴를 선택할 경우, 게임을 시작한다.

4. 2번 메뉴를 선택할 경우, 퀴즈 동안에 풀 총 문제 수를 설정 가능하다.

5. 3번 메뉴를 선택할 경우, 문제 별 답 입력 횟수를 설정 가능하다.

6. 4번 메뉴를 선택할 경우, 퀴즈 동안 사용가능한 총 힌트 횟수를 설정 가능하다.

7. 5번 메뉴를 선택할 경우, 퀴즈에 사용될 소스 파일도 확인 및 수정 가능하다.
(참고)
QuizSource.txt 파일은 
[첫번째줄 : 문제, 두번째줄 : 문제의 정답, 세번째줄 : 문제의 힌트, 네번째줄 : 정답과 관련한 정보] 구성으로 반복된다.

8. 그 후 게임을 즐기고 결과를 확인한다.
